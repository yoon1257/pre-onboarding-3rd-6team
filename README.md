# 프로젝트 설치 및 실행 방법

## ✅ 설치방법

1. Node.JS를 다운받아 설치해주세요.

```
https://nodejs.org/
```

<br/>

2. 리포지토리를 클론해주세요.<br/>

```
git clone https://github.com/yoon1257/pre-onboarding-3rd-6team.git
```

<br/>

3.dependencies를 설치해주세요.

```
npm install
```

<br/>
<br/>

## ✅ 실행방법

1. 다음 명령어를 이용해 react server를 실행시켜주세요.

```
npm run start
```
